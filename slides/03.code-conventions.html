Orden de declaración

  • Variables de clase (static)
  • Variables de instancia
  • Constructores
  • Métodos

Ejemplo:

package com.mycompany.code;

import com.mycompany.util.Constants;

public class Page {
	
	public static int numeroGlobalVisitas;

	private String fileName;
	private Date publishedDate;

	public Page() {
		...
	}

	public void render() {
		...
	}

}

Identación: 4 espacios

Usar en lo posible espacios en lugar de tabs

Tamaño de línea: 80 caracteres

Cuando una expresión no cabe en una línea, se debe romper de acuerdo a los siguientes principios:

Romper después de una coma


method(longExpression1, longExpression2, longExpression3,
    longExpression4, longExpression5);

int value = method1(longExpression1,
                method2(longExpression2, longExpression3));

Romper antes de un operador

Evitar romper expresiones agrupadas por paréntesis


longName1 = longName2 * (longName3 + longName4 - longName5)
                + 4 * longname6; // Ok
longName1 = longName2 * (longName3 + longName4
                - longName5) + 4 * longname6; // Evitar

Declaración por línea

Es preferible una declaración por línea. Esto:

int level;
int size;

Es preferido sobre:

int level, size;

Una sentencia por línea

argv++; argc--; // Evitar!

Sentencias if-else

if (condition) {
    statements;
}

if (condition) {
    statements;
} else {
    statements;
}

if (condition) {
    statements;
} else if (condition) {
    statements;
} else if (condition) {
    statements;
}

Siempre usar llaves

if (condition) // Evitar! No omitas las llaves {}!
    statement;

Convención de Nombres

Clases / Interfaces: UpperCamelCase

Ejemplo:


class Empleado { ... }

class PersonaJuridica { ... }

interface Serializable { ... }

Métodos / Variables: LowerCamelCase


void run() { ... }

int getStockByProduct() { ... }

int nombre;

double sueldoPromedio;

Constantes

Los nombres de las constantes deben ser todo en mayúsculas separados por underscore (_)


final int EDAD_MINIMA = 18;

final String CODIGO_INICIAL = "A00000";

Accediendo a métodos de clase

Evitar usar instancias de la clase para acceder a miembros de clase, en su lugar usar el nombre de la clase.


class MiClase {
    static int variableDeClase = 1;

    static void metodoDeClase() { ... }
}

unObjeto.variableDeClase; // Evitar

MiClase.variableDeClase; // Ok

unObjeto.metodoDeClase(); // Evitar

MiClase.metodoDeClase(); // Ok
					
System.exit(0);
