Instalando Maven

Descarga de Maven (apache-maven-3.3.9-bin.zip)

  1. Crear la carpeta tools en el disco D
  2. Descomprimir el archivo apache-maven-3.3.9-bin.zip en la carpeta tools
  3. Establecer la variable de entorno MAVEN_HOME con el valor D:\tools\apache-maven-3.3.9
  4. Agregar la entrada %MAVEN_HOME%\bin a la variable de entorno PATH
  5. Abrir una ventana de comandos y ejecutar el comando mvn -version
